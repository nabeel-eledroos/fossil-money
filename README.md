This is Fossil Money, a civic transparency web app that helps citizens track fossil fuel donations to their elected officials. [Check it out here!](https://fossil-money.vercel.app)

Purpose: Enter your ZIP code to find your 2 U.S. senators and House representative, see how much fossil fuel money they've accepted, their League of Conservation Voters (LCV) 
climate scores, and whether they've signed the "No Fossil Fuel Money" pledge. Users can then contact their reps with pre-written templates.

It's essentially a "follow the money" tool for climate accountability.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
